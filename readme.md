## Webpack web application
```
npm install
npm run webpack[:production]
```

## Configuration
```
# config/server.config
# host: optional hostname or IP address
# port: optional port
{
  "host": "localhost",
  "port" : 8002
}
```

## Start server
```
npm start # or
bin/server.js
```

## Export vector data
```
bin/export.js
```
